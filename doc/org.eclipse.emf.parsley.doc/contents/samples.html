Code Sections

A code section looks like this:

class HelloWorld {
    public void sayHello(String name) {
        System.out.println("Hello "+name);
    }
}

Note how the code is highlighted according to the language you choose. It is even possible to have different language configurations to get different coloring.

Lists and Tables

Lists are very similar to HTML. We've got ordered lists:

  1. First
  2. Second
  3. Third

and unordered lists:

  • Foo
  • Bar
  • Baz
